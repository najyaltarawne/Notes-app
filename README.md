# 📝 Notes App

* Simple idea of Notes App built using **HTML, CSS, and JavaScript** that allows users to create, edit, delete, and store notes directly in the browser.
  
*Demo: https://najyaltarawne.github.io/Notes-app/
---

## 🚀 Features

* ✍️ Create notes
* 📝 Edit existing notes
* ❌ Delete notes
* 💾 Persistent storage using localStorage (notes remain after refresh)
* 📅 Display creation date for each note
* 🎨 Clean and responsive UI (card-based layout)

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the app
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Logic and interactivity
* **Web Storage API (localStorage)** – Data persistence


---

## 💡 How It Works

* Notes are stored in a JavaScript array
* The array is saved in the browser using **localStorage**
* When the page reloads, saved notes are retrieved and displayed
* Users can perform CRUD operations (Create, Read, Update, Delete)
